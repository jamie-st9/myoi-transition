# Framer Design Patterns

Comprehensive design patterns derived from 100+ Framer marketplace templates.

## 16 Style Categories

### 1. Modern

Definition: Sleek, customizable designs with responsive layouts and sophisticated visual hierarchy.

Key Characteristics:

- Clean, contemporary aesthetics
- Strategic content placement
- Balanced whitespace usage
- Dynamic interaction patterns

Color Approaches: Dark, Light, Colorful, Gradient, Monochromatic options.

Typography: Variable fonts with typographic styling options.

Animation: Scroll effects, sticky scrolling, text effects, 3D transforms, appear effects.

Example Templates: Xtract, Bartix, Saalix, Draftr, Clarity GV, Omega.

### 2. Minimal

Definition: Clean, simple web design emphasizing function over decoration.

Key Characteristics:

- Generous whitespace for breathing room
- Reduced visual clutter
- Content-focused layouts
- Restrained color palettes

Color Approaches: Monochromatic, Light, Black and White, Pastel.

Typography: Modern, legible typefaces (GT Walsheim, Inter, similar).

Animation: Subtle transitions, refined hover states, minimal motion.

Example Templates: Plumbing, Dreelio, Meridian, Sonido, Pearl, Portfolite.

### 3. Dark

Definition: Sophisticated, immersive experiences with mood-focused aesthetics.

Key Characteristics:

- Deep backgrounds (typically #0F172A to #000000)
- High contrast for readability
- Dramatic visual impact
- Premium, exclusive feeling

Color Approaches: Deep grays, pure blacks with vibrant accent colors.

Typography: Light text on dark backgrounds, careful contrast ratios.

Animation: Glow effects, subtle reveals, dramatic transitions.

Example Templates: Cryptix, Etery, MONO AI, Jet, Nitro, JORGE.

### 4. Gradient

Definition: Vibrant color transitions creating modern depth and visual interest.

Key Characteristics:

- Multi-color gradient flows
- Mesh gradient backgrounds
- Dynamic color interactions
- Depth through color blending

Gradient Types:

- Linear: Directional color transitions
- Radial: Circular color emanation
- Mesh: Complex multi-point gradients
- Animated: Motion-based color shifts

Example Templates: Fusion AI, Limitless, Creditflow, Fluence AI, Waitlister, EarlyBird.

### 5. 3D

Definition: Depth, perspective, and immersive visual experiences.

Key Characteristics:

- Three-dimensional elements and transforms
- Parallax scrolling effects
- Depth shadows and layering
- Perspective-based layouts

Technical Features:

- 3D transforms for rotation and depth
- Z-axis positioning for layering
- Shadow techniques for dimension
- Perspective adjustments for immersion

Example Templates: Portfolite, Orbai, LanderOS, Sadewa, Quorum, Blocks.

### 6. Animated

Definition: Motion-driven storytelling with attention-guiding interactions.

Key Characteristics:

- Purposeful motion design
- Scroll-triggered animations
- Micro-interactions
- Performance-conscious effects

Animation Types:

- Scroll effects: Triggered by scroll position
- Appear effects: Entrance animations
- Text effects: Typography animations
- Sticky scrolling: Fixed position during scroll range
- Custom cursors: Interactive cursor changes

Example Templates: Xtract, Fusion AI, Handyman, Palmer, Nitro, Artemis.

### 7. Light

Definition: Bright, airy designs with predominantly white or light backgrounds.

Key Characteristics:

- White or off-white backgrounds
- Dark text for high contrast
- Clean, open feeling
- Professional, trustworthy appearance

Color Applications:

- Background: #FFFFFF to #FAFAFA
- Text: #111827 to #374151
- Accents: Brand colors for highlights

### 8. Colorful

Definition: Vibrant, multi-color palettes creating energetic visual experiences.

Key Characteristics:

- Bold color combinations
- Playful, energetic feeling
- Creative expression through color
- Attention-grabbing visuals

Color Strategies:

- Complementary: Opposite colors for contrast
- Triadic: Three evenly spaced colors
- Analogous: Adjacent colors for harmony
- Split-complementary: Main color with two adjacent complements

### 9. Monochromatic

Definition: Single-color family with varying shades, tints, and tones.

Key Characteristics:

- Visual cohesion through color unity
- Elegant, sophisticated appearance
- Focus on content over color
- Easy brand alignment

Implementation:

- Choose base hue
- Create variations through saturation and lightness
- Use neutral gray for balance
- Apply strategically for hierarchy

### 10. Pastel

Definition: Soft, muted colors creating gentle, approachable aesthetics.

Key Characteristics:

- Low saturation colors
- Calming, friendly feeling
- Often combined with minimal design
- Suitable for wellness, lifestyle, education

Common Palettes:

- Soft pink, lavender, mint
- Cream, peach, sky blue
- Sage green, soft coral, pale yellow

### 11. Black and White

Definition: Pure contrast design using only black, white, and gray tones.

Key Characteristics:

- Maximum contrast
- Timeless, classic appearance
- Focus on typography and form
- Dramatic visual impact

Applications:

- Portfolio sites
- Editorial designs
- Luxury brands
- Photography showcases

### 12. Grid

Definition: Structured layouts based on systematic grid frameworks.

Key Characteristics:

- Consistent alignment
- Organized information hierarchy
- Balanced visual weight
- Scalable responsive design

Grid Systems:

- 12-column for flexibility
- 8-column for focused layouts
- Custom grids for unique needs
- Asymmetric grids for creative expression

### 13. Illustrative

Definition: Design incorporating custom illustrations and graphics.

Key Characteristics:

- Unique visual identity
- Storytelling through imagery
- Brand differentiation
- Emotional connection

Illustration Styles:

- Line art for minimal aesthetic
- Flat design for modern look
- Isometric for technical products
- Hand-drawn for organic feeling

### 14. Large Type

Definition: Typography-forward design with oversized text elements.

Key Characteristics:

- Statement headlines (72px+)
- Typography as visual element
- Bold creative expression
- Immediate attention capture

Implementation:

- Hero headlines: 72-120px
- Section titles: 48-72px
- Variable font weights
- Careful line height management

### 15. Retro

Definition: Design inspired by past eras (70s, 80s, 90s, Y2K).

Key Characteristics:

- Nostalgic color palettes
- Period-appropriate typography
- Vintage visual elements
- Creative interpretation of past styles

Era References:

- 70s: Earth tones, organic shapes
- 80s: Neon, geometric patterns
- 90s: Grunge, bold primary colors
- Y2K: Metallic, futuristic, chrome

### 16. Professional

Definition: Business-focused design emphasizing trust and credibility.

Key Characteristics:

- Conservative color choices
- Clear information hierarchy
- Trustworthy appearance
- Conversion-optimized layouts

Applications:

- B2B SaaS products
- Financial services
- Consulting firms
- Enterprise software

## Implementation Guidelines

### Starting a New Design

Step 1: Determine primary style category based on brand and audience.

Step 2: Select 2-3 complementary style tags from Pencil.

Step 3: Load style guide with get_style_guide for selected tags.

Step 4: Plan section structure following conversion-optimized patterns.

Step 5: Implement sections progressively with batch_design.

Step 6: Validate each major section with get_screenshot.

Step 7: Iterate based on visual feedback.

### Combining Styles

Effective Combinations:

- Modern + Dark + Gradient: Tech startups, AI products
- Minimal + Light + Professional: B2B SaaS, enterprise
- Colorful + Animated + Bold: Creative agencies, entertainment
- Dark + 3D + Animated: Gaming, crypto, immersive products
- Minimal + Pastel + Light: Wellness, lifestyle, education

Avoid Conflicting Combinations:

- Minimal + Colorful + Animated (too much contrast in approach)
- Retro + Modern (conflicting era references)
- Professional + Playful (mixed tone signals)
